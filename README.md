## 运行

1. 安装项目依赖

```bash
npm install

// 或
yarn 
```

2. 安装 API 服务

API service 来自 [https://github.com/Binaryify/NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi)

```
// 安装
git clone https://github.com/Binaryify/NeteaseCloudMusicApi.git

npm install

// 运行
node app.js
```
服务器启动默认端口为 3000
